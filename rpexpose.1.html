NAME

RpExpose − The Ratpoison Graphical Pager

SYNOPSIS

rpexpose −−help | −−version
rpexpose −−generate [file]
rpespose −−select [file]
rpexpose −−clean

EXAMPLES

From rpthumb
ratpoison −c “fdump” |
tr “,:” “\n\n” |
sed “s/[()]//g” |
awk ´/window/ {print $2}´ |
rpexpose −−generate −

From rpselect
ratpoison −c "windows %s:%i:%n:%t" |
sed −e “s/^*:/1:/” −e “s/^[+−]:/0:/” |
rpexpose −−select −

Add the following lines to your .ratpoionrc file
addhook switchwin exec rpthumb
addhook quit exec rpexpose --clean
bind [key] exec rpselect

DESCRIPTION

RpExpose is a graphical pager written for the Ratpoison window manager, but designed to be used with any window manager.

OPTIONS

−h −−help

Show summary of options

−v −−version

Show the version of program

−g −−generate file

Generates a thumbnail image of the top level windows listed in file. file should list the x window id´s of all of the top level windows. If file is −, then rpexpose will read from stdin. Assuming that you are using Ratpoison, rpthumb (provided with rpexpose) is a prewritten wrapper around −−generate, and you should simply call that instead.

−s −−select file

Creates a dialog box with thumbnails of all of the open windows and allows you to select one to raise. file contains a list of the current windows the syntax of which is shown below in FILES. Assuming that you are using Ratpoison, rpselect (provided with rpexpose) is a prewritten wrapper around −−select, and you should simply call that instead.

−c −−clean

Cleans all of the image files in the ~/.rpexpose/ directory. This function should be called when your window manager exits to ensure that the ~/.rpexpose/ directory does not fill up with garbage data.

FILES

−s −−select file

Each line of file must follow the following format: [1_if_selected,_0_otherwise]:[X_Window_ID]: [Internal_ID]:[Window_Name]

~/.rpexposerc

A sample rpexposerc is included along side rpexpose for your convenience. The rpexposerc file is configured in a similar manner to Ratpoison’s rc file. The commands are described in the following section.

RPEXPOSERC FILE

left (h j) | right (l k)

Select the left or right thumbnail

quit (q)

Quits rpexpose

select (Space Return)

Raises the current window and exits rpexpose

bind [key] [command]

Binds the key to the given command. Execute command every time key is pressed

unbind [key]

Unbind the key. Remove the action associated with key

exec [command]

Execute the supplied command

set [option] [value]

Set the internal option to the supplied value. A list of options is supplied in the following section

RPEXPOSE OPTIONS

widescreen [boolean]

rpexpose attempts to make the selection window fit the shape of the screen as closely as possible. If you have a wide-screen monitor, set this to be true

thumb_padding [integer]

thumb_padding is the space between each thumbnail, as well as the thumbnail and the edge of the window.

text_padding [integer]

text_padding is the whitespace padding around each of the numbers in the top left hand corner of the thumbnails.

border_padding [integer]

border_padding is the width of the border around the thumbnail of the selected window.

select_exec [string]

select_exec is set to the string to be executed in order to raise a window. The values %x, %n, and %i are replaced with the X_Window_ID, Window_Name, and Window_ID respectively.
