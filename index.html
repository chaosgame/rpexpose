RpRestart

RpRestart is a simple session manager. Along with the following two lines in your .ratpoisonrc, it dumps and restores the frames across restarts. The trick here is that newwm ratpoison does the same thing as the default restart, so we can alias restart to run a script, and then call newwm ratpoison from within the script.

~/.ratpoisonrc

addhook quit exec echo "" > ~/.ratpoison_frames
exec ratpoison -c "frestore `cat ~/.ratpoison_frames`"
alias restart exec rprestart

RpRestart

#!/bin/bash
ratpoison -c "fdump" > ~/.ratpoison_frames
ratpoison -c "newwm ratpoison"

RpVimmode

RpVimmode toggles h,j,k,l between being h,j,k,l and up,right,left,down.

RpVimmode

#!/bin/sh
if [ -e ~/.rpvimmode ]; then
echo "
keycode 43 = h H
keycode 44 = j J
keycode 45 = k K
keycode 46 = l L" |
xmodmap -;
ratpoison -c "echo Vim Mode Off"
rm ~/.rpvimmode;
else
echo "
keycode 43 = Left H
keycode 44 = Down J
keycode 45 = Up K
keycode 46 = Right L" |
xmodmap -;
ratpoison -c "echo Vim Mode On"
touch ~/.rpvimmode;
fi
